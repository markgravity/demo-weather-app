# Aurora Weather

A beautiful weather app built with SwiftUI and WeatherKit for iOS and macOS.

## Features

- Real-time weather data via WeatherKit
- 10-day forecast with hourly breakdowns
- Interactive radar maps
- Severe weather alerts
- Home Screen widgets
- Native macOS menu bar companion

## Requirements

- iOS 17.0+ / macOS 14.0+
- Xcode 16.0+
- Swift 6.0+
- WeatherKit entitlement

## Getting Started

```bash
git clone https://github.com/markgravity/demo-weather-app.git
cd demo-weather-app
open Package.swift
```

## Architecture

The app follows a clean MVVM architecture with SwiftUI and Swift Concurrency.
